El fuelle de fragua

En las fraguas el herrero necesitaba tener una corriente de aire constante para poder trabajar con el hierro. Los fuelles normales cogen aire para luego soplarlo, pero cuando cogen aire se corta el soplido. El fuelle de fragua se inventó para solucionar este problema. Son dos fuelles juntos colocados para que uno sople mientras el otro coge aire, proporcionando una corriente de aire constante.
